# Element-UI 类似的组件库

这是一个类似于 Element-UI 的组件库，包含了以下八个组件：

- Alert
- Button
- Collapse
- Message
- Notification
- Popconfirm
- Tooltip

## 使用方法

组件的使用方法可以在以下目录找到：
packages/docs/Components


## Demo 演示

### 工具组件演示

Alert, Button, Collapse, Tooltip 四个组件的演示可以通过以下步骤进行：

1. 进入 `play` 目录。
2. 运行 Storybook 以查看演示。

### 其他组件演示

剩余组件的演示可以在以下目录找到每个组件的 Demo：
packages/docs/demo


要展示这些组件，请将相应的 Demo 复制到 `play` 目录下的 `app` 文件夹中。
